# 바른라운지 고객 LMS 템플릿 초안 (3종)

- **작성일:** 2026-07-21 · 백엔드(발송 서버) 구현용
- **발송 유형:** 전건 거래 정보성 메시지 — (광고) 표기·수신거부 문구 불필요, 광고 수신 동의와 무관하게 발송 가능
- **정책 반영:** 방문 전 예약 필수 · 유효기간 연장 불가 · 환불은 고객 신청 기반(자동 환불 없음) · D-30 통지 의무(약관 제7조 ②)

---

## ① 구매 완료 LMS — 결제·발급 즉시 발송

```
[바른라운지] 이용권 구매 완료 안내

{고객명}님, 이용권 구매가 완료되었습니다.

■ 상품명: {상품명}
■ 쿠폰번호: {쿠폰번호}
■ 유효기간: {만료일}까지
■ 사용 가능 지점: {사용가능지점}

· 방문 전 지점 예약은 필수입니다. 반드시 사전 예약 후 방문해 주세요.
· 매장 방문 시 쿠폰번호 또는 구매자 이름·연락처를 제시해 주세요.
· 쿠폰번호는 마이페이지 > 쿠폰함에서 언제든 확인할 수 있습니다.
```

## ② 유효기간 만료 30일 전 LMS — D-30 자동 발송 (약관상 통지 의무)

```
[바른라운지] 이용권 유효기간 만료 30일 전 안내

{고객명}님, 보유하신 이용권의 유효기간이 30일 남았습니다.

■ 상품명: {상품명}
■ 쿠폰번호: {쿠폰번호}
■ 남은 잔액/횟수: {잔여표시}
■ 만료일: {만료일}
■ 사용 가능 지점: {사용가능지점}

· 유효기간은 연장되지 않으니 기간 내 사용해 주세요. (방문 전 지점 예약 필수)
· 사용이 어려운 경우 마이페이지 > 쿠폰함에서 환불을 신청하실 수 있습니다.
  (미사용 시 결제금액 100%, 일부 사용 시 잔여분 결제금액 비례)
· 만료 후에도 구매일로부터 5년 이내 신청 시 잔여분 상당 결제금액의 90%를 환급해 드립니다.
· 환불은 자동으로 처리되지 않으며, 신청하신 경우에만 처리됩니다.
```

> ⚠️ 축약하더라도 **잔액·만료일·환불 방법 3요소는 유지** — 약관 제7조 ② 통지 의무 요건.

## ③-A 사용 완료 LMS — 일반형 사용 / 금액·횟수형 잔여 소진 시

```
[바른라운지] 이용권 사용 완료 안내

{고객명}님, '{상품명}' 이용권이 사용 완료되었습니다.

■ 사용 지점: {사용처명}
■ 사용 일시: {사용일시}

이용해 주셔서 감사합니다.
· 본인이 사용하지 않으셨다면 고객센터({고객센터번호})로 즉시 연락해 주세요.
```

## ③-B 부분 사용 LMS — 금액형 차감 / 횟수형 회차 사용 (잔여 남음)

```
[바른라운지] 이용권 사용 안내

{고객명}님, '{상품명}' {차감표시}가 사용되었습니다.

■ 사용 지점: {사용처명}
■ 사용 일시: {사용일시}
■ 남은 잔액/횟수: {잔여표시}

· 남은 금액(횟수)은 유효기간({만료일}) 내 사용할 수 있습니다. (방문 전 지점 예약 필수)
· 본인이 사용하지 않으셨다면 고객센터({고객센터번호})로 즉시 연락해 주세요.
```

> ③ 분기 기준: 사용 후 잔여(balance) = 0 → **③-A**, 잔여 > 0 → **③-B** (usages의 usageType·balance로 판단)

- "본인 미사용 시 고객센터 연락" 문구는 부정 사용 조기 탐지용 — 유지 권장.

- 관련 화면 문구(상품 상세 하단 안내사항)와 동일 정책 기준으로 작성됨.
